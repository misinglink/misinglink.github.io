#set point coordinates
x =[0,.5,1]
y = [0,np.sqrt(.75),0]
ry = 0
rx = np.random.rand(1,)
start = [ry, rx]
a, b, c = zip(x, y)

#set list of vertices for random choice
direction = [a,b,c]

#function to pick a random point in the triangle
def rand_dir(dirc):
    return np.array(random.choice(dirc))

#plot triangle
plt.figure(figsize=(10, 10))
plt.scatter(x, y)

#plot initial random point
plt.scatter(rx, ry)

#this loops makes sure different distances to the NEXT 
#RANDOMLY CHOSEN point
for f in range(1, 11):
    
    #this loop define the amount of iterations(points) in the picture
    for i in range(10000):
        dirc = rand_dir(direction)
        start = (dirc + start) * (f/10)
        point = plt.scatter(start[0], start[1], s = 5)
    
    #the picture title d= is the distance to the next
    #randomly chosen point
    # and  p= the amount of iterations in the picture
    plt.savefig('ChaosGameTriangleD='+str(f)+'_P='+str(10000)+'.png')


    plt.show()

   I first started this 'experiment' very playfully and out of curiosity. i simply wanted to see if I could recreate the simple algorithm (pick a random point if the shape, go halfway to that point, mark a point and repeat) that is the Chaos Game with what I was learning in Python. Turns out it was pretty easy. 
    The next question was, why does order arise in this model? Does it have to do with the distance travelled to the random point? So I changed the distance travelled and decided to print out all the pictures to see how they compared. Turns out before d=.5  there is order, but anything above it slowly starts becoming more chaotic, until a different kind of order arises, a striaght line that kind of resembles y=x. I found fascinating. The same process produces vastly different results. Can I measure chaos or variability based on the area that the point cover? 
    As you can tell this project is full of tangents that one might want to explore. Did you notice the area of the attractive sites starts to increase as d increases? What happens when you add vertices to the shape?
